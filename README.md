s12x-build
==========

**s12x-build** contains a script to build binutils, GCC and newlib for 68HC11, 68HC12, S12, S12X and XGATE microcontrollers.

Getting Started
---------------

```bash
export PATH="`pwd`/dist/bin:$PATH"
make
```
